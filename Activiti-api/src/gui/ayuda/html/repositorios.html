Pantalla de seleccion de repositorio

En esta apantalla aparece un campo de texto para introducir el nombre de un usuario cualquiera registrado en la plataforma para obtener los repositorios de los que es propietario y cargarlos en la lista existente mas abajo.
El boton "Mostrar resultados" carga la pantalla resultados para mostrar las metricas calculadas para el repositorio seleccionado.
