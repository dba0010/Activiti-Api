PorcentajeIssuesCerradas

Propósito: ¿Proporcion de issues cerradas en el repositorio en funcion de las creadas?
Formula: PIC = NIC (Numero de issues cerradas) * 100 / NI (Numero de issues)
Interpretación: 0 <= N <= 100 mejor valores altos
Tipo de escala: Ratio
Tipo de medida: NIC contador, NI contador
Fuente de medición: Repositorio GitHub de un proyecto
