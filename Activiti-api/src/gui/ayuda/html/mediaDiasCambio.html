MediaDiasCambio

Propósito: ¿Cuantos dias de media pasan entre cambios?
Formula: MDC = D (Dias desde el primer al ultimo cambio) / NTC (Numero total de cambios)
Interpretación: MDC >= 0 mejor valores bajos
Tipo de escala: Absoluta
Tipo de medida: D contador, NTC contador
Fuente de medición: Repositorio GitHub de un proyecto
